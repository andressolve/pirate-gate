# Pirate Gate: One Piece

**Francisco's game studio build.**

A 3D anime cel-shaded action game set in the One Piece universe. Sail the Grand Line as Luffy, fight Marines with Devil Fruit powers, and race to extract the One Piece.

## Mission Briefings

- [Mission 1: Forge Your Captain](https://andressolve.github.io/pirate-gate/missions/mission-1-forge-your-captain.html) — Design Luffy from concept art to animated 3D fighter

## Studio Build Plan

See [pirate-gate-studio-plan.md](pirate-gate-studio-plan.md) for the full session-by-session roadmap.

## The Vision

Four pirate factions. Asymmetric abilities. Ship combat and ground combat. Devil Fruit risk-reward. The One Piece as the extraction objective.

Read the full vision: [vision_f.md](vision_f.md)
