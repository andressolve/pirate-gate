Pirate
Gate Francisco's Game Studio

A 3D anime cel-shaded action game set in the One Piece universe. Built session by session with Claude Code — designed by Francisco, powered by AI.

The Designer's Vision
Four pirate factions race across the Grand Line to find the One Piece. Pick a captain — get their ship, their crew, their Devil Fruit powers. Luffy stretches. Arlong swims. Buggy splits apart. Your ship is your life — if it sinks, your crew is eliminated. Sail, fight, explore, extract. Every match tells a different story.
Tech Stack
Engine

Three.js

3D browser game. Cel-shaded anime rendering. Deploys as a shareable URL.

3D Models

Meshy MCP

Image-to-3D generation, auto-rigging, 500+ animation presets. All inside Claude Code.

Concept Art

Gemini MCP

AI image generation for character concepts and clean Meshy-ready reference images.

IDE

Claude Code

The entire game is built through conversation. Francisco describes, Claude builds.
