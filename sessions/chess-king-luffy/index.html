The King: Luffy Chess Piece

Design and 3D print a King chess piece featuring Monkey D. Luffy. Everything happens in Claude Code.

By the end of this session, you'll have a print-ready STL file of a Luffy King chess piece that you designed.

1 Download the session file

This file tells Claude Code everything it needs to know to run the session with you — how to generate concept art, convert it to 3D, and prepare it for printing.

Download CLAUDE.md

What is this file? It's instructions for Claude Code. When Claude Code opens in a folder that has a CLAUDE.md file, it reads it automatically and knows what to do. You don't need to read it yourself — just put it in the right place.

2 Put it in the right folder

Create a new folder for this session, then put the CLAUDE.md file inside it:

Documents/ francisco-game-studio/ sessions/ chess-king-luffy/ ← put CLAUDE.md here CLAUDE.md

If the folder already exists, just drop the file in. If not, create it.

The file must be named exactly CLAUDE.md (capital letters, no spaces). Claude Code only reads it if the name is exact.

3 Open Claude Code in that folder

Open your terminal and run:

cd ~/Documents/francisco-game-studio/sessions/chess-king-luffy claude

Claude Code will start up and automatically read the CLAUDE.md. It knows the mission.

First time? Claude Code might need to set up the Meshy and Gemini connections. If it says something about "MCP not found" or asks your parent for an API key, that's normal — it just needs a quick setup. The instructions are in the CLAUDE.md.

4 Start talking

Claude Code will greet you and set the mission. Then it's a conversation. Here's what you'll do:

  1. Describe your Luffy King. Tell Claude what you see — the pose, the outfit, the straw hat, the base. Claude asks questions to sharpen the picture. (~10 min)
  2. Generate concept art. Claude creates images based on your description. You say what's right and what to change. Iterate until it looks like YOUR Luffy. (~15 min)
  3. Convert to 3D. Claude sends your concept art to Meshy and builds a 3D model. You'll see it in a browser viewer you can rotate and check from every angle. (~10 min)
  4. Prep for printing. Claude resizes the model, exports it as STL, and repairs the mesh. You get a file ready to open in QIDIStudio and print. (~10 min)

Total time: ~45 minutes. At the end you'll have a print-ready STL of a King chess piece that you designed. If you have time left, you can design another piece (Queen? Pawn? The enemy King?).

Things to think about before you start

You don't need to decide everything now — Claude will ask you during the session. But it helps to have some ideas:

  • Which Luffy? Pre-timeskip (red vest, blue shorts)? Post-timeskip (open shirt, chest scar)? Gear Fifth? Your own version?
  • The straw hat. On his head? Tilted? Around his neck? This is the crown of the King piece.
  • Pose. Standing tall like a captain? Arms crossed? Fist up? Remember: arms close to body print better.
  • The base. Classic round chess pedestal? Ocean waves? Skull and crossbones? Something else?
  • How tall? Standard King is ~9 cm. Bigger for a display piece? Smaller for a travel set?
